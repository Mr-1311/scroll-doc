Pages

Variables

In org documents user can define some variables to use it on templates or change some config for only this file.

Variable names are case insensitive.

NOTE: More variables will come soon.

title

Can be used in templates.

Usage:


#+TITLE: Page Title

Date

Can be used in templates.

Usage:


#+DATE: 2020-01-01

Template

Template variable determine template to use for this page.

Usage:


#+TEMPLATE: my_template.html
